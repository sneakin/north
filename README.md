Building
---

You will need `git`, `node.js`, `ruby`, `rake`, and a web browser.

1. git submodule --init
2. npm install
3. rake

Running
---

* Open `build/runner.html`
* Run Bacaw's bccon or development page using one of the BIN files in the build directory.
